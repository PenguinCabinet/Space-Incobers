# 👾 Space-Incobers

https://github.com/user-attachments/assets/939306db-dac4-4c23-90bb-f3a1d43adbe4

A Space Invaders-like game written in COBOL.It runs in the terminal.

COBOLで書かれたスペースインベーダー風のゲーム。ターミナルで動作します。

# Usage

```
cobc Space-Incobers.cob
```
```
./Space-Incobers
```
Press the W key to fire a bullet.      
Press the A key to move left, and the D key to move right.      
After a game over, press the Enter key to exit the game.
